Project Title
E-Commerce Sales & Customer Analysis

Objective
Analyze e-commerce sales data to understand revenue performance, customer behavior, cancellations, and delivery efficiency, and present insights through an interactive Power BI dashboard.

Tools Used

SQL (MySQL)

Power BI

Excel

Key Analysis Performed

Total Revenue and Average Order Value (AOV)

Cancelled Orders and Revenue Impact

Late Delivery Percentage and delivery performance

Customer Lifetime Value (CLV)

Repeat vs One-time customer contribution

Product category-wise revenue

Monthly revenue trend analysis

Data Modeling

Built fact and summary tables using SQL for optimized BI reporting

Created clean relationships for accurate aggregation in Power BI

Dashboard Features

KPI cards for revenue, AOV, cancellations, and delivery metrics

Category-wise and time-based revenue trends

Customer segmentation (Repeat vs One-time)

Year-based slicers for dynamic analysis

Outcome
The project demonstrates strong SQL querying, data modeling, KPI analysis, and Power BI dashboard development skills for real-world business decision-making.
